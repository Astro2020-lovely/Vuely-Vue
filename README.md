## Set Environment Variable - frontend or vront repository (Important)

```
VUE_APP_API_URL
```

Example
```
VUE_APP_API_URL=htpps://api.helal.zntra.net
```

## Set Environment Variable - backend or back repository  (Important)

```
NODE_ENV=dev
PORT=8000
MONGO_DB_NAME=***
MONGO_DB_USER=***
MONGO_DB_PASS=***
JWT_SECRET=***
```
